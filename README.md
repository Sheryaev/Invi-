# Инви — инвестиционный дашборд.

Персональный дашборд для отслеживания инвестиционного портфеля через T-Invest API (Т-Банк). Показывает стоимость портфеля, доходность, пассивный доход по дивидендам и купонам, ближайшие выплаты, топ роста и падений за день.

## Стек

- **Next.js 14** (App Router)
- **React 18**
- **T-Invest API** — данные о портфеле, ценах, дивидендах и купонах в реальном времени

## Запуск

### 1. Установить зависимости

```bash
npm install
```

### 2. Запустить в режиме разработки

```bash
npm run dev
```

Приложение откроется на [http://localhost:3000](http://localhost:3000).

### 3. Подключить портфель

При первом открытии нужно ввести токен T-Invest API:

1. Получи токен в приложении Т-Инвестиции: **Профиль → Настройки → Токен для T-Invest API**
2. Введи токен в поле на экране приветствия
3. Выбери нужный счёт из списка

Токен хранится только в браузере (localStorage) и никуда не отправляется кроме T-Invest API.

## Сборка для продакшена

```bash
npm run build
npm run start
```
